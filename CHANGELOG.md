# Changelog

## [2.0.9] - 2026-04-16

### Fixed

- 修改严格模式

- 所有可能失败的操作都加了容错

- 改进输出格式

- 增强扩展名提取逻辑

- 增强日志容错

### Added

- 添加文件存在性检查

## [2.0.8] - 2026-04-16

### Fixed

- 修复关联数组遍历 (第 337 行附近)：

- 添加去重逻辑：使用关联数组 unique_map 对目录去重

- 改进扩展名检查：使用 [[-v "RULES[$ext]"]] 检查键是否存在

- 版本更新到 2.0.8

## [2.0.7] - 2026-04-16

### 修复

- 修复 `organize_directory` 函数中关联数组导致的 "未绑定的变量" 错误
  - 将关联数组改为普通数组，避免在 `set -u` 严格模式下的崩溃
  - 影响范围：当没有需要创建的目录时，脚本会正常退出而不会报错

### 优化

- 改进目录创建逻辑，使用普通数组存储待创建目录列表
- 增强错误处理，避免因空数组导致的脚本中断

## [2.0.6] - 2026-04-16

### Fixed

- 修复终端颜色代码显示问题（颜色代码被直接打印而非渲染）
- 使用 printf 替代 echo -e 提高兼容性
- 改进颜色检测逻辑，优先使用 tput
