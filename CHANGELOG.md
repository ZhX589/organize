# Changelog

## [2.0.1] - 2026-04-16

### Added

- 添加 GitHub Actions 自动构建

## [2.0.0] - 2026-04-16

### Added

- 支持 deb、rpm、AUR 打包
- 新增 `--init` 选项创建默认规则
- 新增 `--edit-rules` 选项
- 新增 XDG 基目录规范支持
- 添加详细的手册页

### Changed

- 重构代码，模块化函数
- 规则文件格式改为 `扩展名:目标目录`（支持多个扩展名逗号分隔）
- 优化输出，彩色显示

### Fixed

- 修复文件冲突时跳过逻辑
- 修复无扩展名文件的处理

## [1.0.0] - 2026-04-01

### Added

- 初始版本
- 基本整理功能
- 规则管理命令
